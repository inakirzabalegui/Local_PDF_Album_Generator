Guía de Uso

¿Cómo funciona?

La pestaña Fuente te permite revisar, renombrar y borrar eventos y fotos del origen antes de generar el álbum.

La pestaña Edición te permite reordenar fotos, cambiar títulos, seleccionar layouts y reorganizar páginas del workspace.

Flujo recomendado

  1. Abre o arrastra la carpeta de fotos con 📁 Abrir carpeta
  2. Revisa los eventos en Fuente: renombra y borra lo que no quieras
  3. Pulsa 🔄 Regenerar Álbum para crear el workspace de páginas
  4. Cambia a Edición para ajustar fotos, títulos y layouts
  5. Guarda con Cmd/Ctrl+S y exporta el PDF desde la línea de comandos

Acciones en Fuente

  • ✏️ Renombrar Evento — cambia el nombre del evento y renombra automáticamente las fotos con el nuevo prefijo
  • 🗑️ Borrar Evento — elimina la carpeta de evento completa
  • 🗑️ Borrar Foto — elimina la foto seleccionada del origen
  • ✅ Completado — marca el evento como revisado (punto verde)
  • 🔄 Regenerar Álbum — reconstruye el workspace de páginas desde cero

Acciones en Edición

  • ✏️ Renombrar Página — edita el título de sección
  • 📝 Renombrar Subtítulo — edita el subtítulo de la página
  • 🎨 Modo Layout — elige entre Mesa de Luz, Grid Compacto o Híbrido
  • 💥 Explotar Página — divide la página en dos (útil si hay demasiadas fotos)
  • 🗑️ Borrar Página — elimina la página completa
  • 🗑️ Borrar Foto — elimina la foto seleccionada de la página
  • ✅ Completado — marca la página como revisada
  • ↕️ Reordenar — arrastra fotos en el panel lateral para cambiar su orden

Cabecera global

  • 📁 Abrir carpeta — abre un selector nativo para cambiar de álbum
  • 🔄 Regenerar Álbum — disponible desde cualquier pestaña
  • ↶ Deshacer — revierte el último cambio en Edición
  • 💾 Guardar — guarda todos los cambios pendientes
  • ✕ Salir — cierra el editor
  • ES/EN Idioma — alterna entre español e inglés
  • { } Debug — muestra el log de depuración interno
